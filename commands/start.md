# Start

Jalankan `.\scripts\start.ps1` — auto semua:

1. Auto-setup (clone ECC & 9Router jika belum)
2. Start 9Router
3. Combo setup — pilih combo yang udah dibuat di dashboard
4. Apply profile ke ~/.config/opencode/
5. Cek update & sync changelog
6. Buka opencode
