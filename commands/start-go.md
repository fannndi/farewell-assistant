# Start Go

Morning routine dengan model Go.

Flow:
1. Cek 9Router running → start kalau belum
2. Cek ECC available
3. Apply go profile
4. Cek LLM mode → start Ollama kalau mode on
5. Print summary

Jalankan: `.\scripts\start.ps1 -Profile go`
